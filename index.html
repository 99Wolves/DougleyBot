DougleyBot, the Discord bot made for communities of all categories.

Meet DougleyBot, a Discord bot created for fun and silliness.

Currently used in more than 20 different Discord servers, approximately 100 users are already using the ready-to-use version of the bot to make their chats a bit more chaotic fun!

Making use of the excellent Discord.js library made by the awesome @hydrabolt, the bot is able to handle a sustantial ammount of users, and a lot of simultanious commands.

Free to use, distribute, and alter.

DougleyBot is created with the GNU license with the source code located at GitHub, making it completely open-source.

If you aren't sure about redistributing, see this paragraph as a super official permission slip to distribute and alter the code as much as you see fit, it's open-source, so go nuts!

Ready-to-use, no installation needed.

DougleyBot also hosts a server on which the ready-to-use version is located.

Because a already live version is present, users don't have to worry about hosting their own server, dealing with downtime, crashes and other annoyances!

And it's completely free to use the live version, no removed commands, paywalls, donation requests, DLC or any other stupid thing that will make it annoying to have the bot in a server!

Interested? Join the test server!

Active development and support.

The devs of DougleyBot are always trying to improve the code, make new functions and other fun stuff!

By the way, because we're open-source, you can make a request to improve our code via GitHub with a pull request, we will gladly make you do our work! we can always use help!

Have a question? Send us a message through the test server, and we will help you to the best of our abilities!
